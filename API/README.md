# Reverse-Jam

Ðis projektas yra paprasta **.NET Web API** sistema su vartotojø registracija, prisijungimu ir Google sign-in. Projektas naudoja **JWT** autentifikacijai.

## Kaip paleisti projektà
1. Klonuok repo:
```bash
git clone https://github.com/Arbatininkai/Reverse-Jam.git
```
2. Eik á projekto katalogà:
```bash
cd Reverse-Jam/API
```
3. Atidaryk projektà su Visual Studio
```bash
-[ ] Atidaryk API.sln per Visual Studio.
```
4. Paleisti projektà:
```bash
-[ ] Paspausk F5 arba CTRL + F5.
```
5. API pasiekiama adresu:
```bash
http://localhost:5000 arba https://localhost:5001
```
